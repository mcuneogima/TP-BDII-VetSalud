# TP-BDII-VetSalud

## Scripts disponibles

```bash
npm start
npm run check
```

### Qué hace cada uno

- `npm start`: levanta el proyecto, conecta MongoDB y Cassandra, y deja todo listo.
- `npm run check`: verifica que ambas conexiones respondan correctamente.

### Requisitos

- MongoDB disponible en `mongodb://localhost:27017`
- Cassandra disponible en `127.0.0.1` con `localDataCenter: datacenter1`
